## 欢迎Pr

欢迎修改 `anime.csv` 来补充或调整名单  

每个条目需要维护 `category`、`score`、`name`、`Eng_name` 和 `reason` 五列  
其中 `Eng_name` 只用于生成英文 `LICENSE`，不会显示在中文 README 中  
提交 PR 时仅需要改这一个文件就行，`README.md` 的表格、`LICENSE_CN` 和 `LICENSE` 的名单都由 Github Action 自动维护

让我们一起建设伟大的反二次元内鬼吧吧.jpg
